# 機能一覧

ユーザーが何をできるかの観点で整理する。技術的な実装詳細はここには含めない。

## 初期フェーズ（GitHub）

### トレンドリポジトリ一覧

- GitHub で注目されているリポジトリの一覧を閲覧できる
- リポジトリ名、説明、スター数、言語、作者などの基本情報を確認できる

### 言語別フィルタリング

- プログラミング言語でトレンドを絞り込める
- 複数言語の選択・解除ができる

### 期間別表示

- 日次・週次などの期間でトレンドを切り替えて表示できる

### トレンド比較

- 複数のリポジトリやトレンド項目を並べて比較できる
- 注目度の相対的な位置づけを視覚的に把握できる

## 将来フェーズ

### 複数ソース横断閲覧

- GitHub、Reddit、Hacker News など複数ソースのトレンドを一画面で確認できる
- 同一技術が複数ソースで言及されている場合に集約表示される

### 時系列トレンド推移

- 特定の技術やリポジトリの注目度推移をグラフで確認できる
- 一過性のバズと持続的なトレンドを視覚的に区別できる

### 通知・アラート

- 特定技術の急上昇を検知して通知を受け取れる
- ウォッチ対象を設定し、変化があった際に知らせてもらえる

### ウォッチリスト

- 関心のある技術やリポジトリをウォッチリストに登録できる
- ウォッチリストに基づいたパーソナライズされたトレンド表示ができる
